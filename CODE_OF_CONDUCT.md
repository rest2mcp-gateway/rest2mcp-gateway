# Code of Conduct

## Our Pledge

We want this project to be an open, respectful, and constructive space for everyone involved.

Participants in this project are expected to:

- be respectful in communication
- assume good intent, while staying technically rigorous
- welcome constructive feedback
- focus criticism on ideas, code, and behavior, not people
- avoid harassment, discrimination, and abusive conduct

## Expected Behavior

Examples of acceptable behavior include:

- giving clear and respectful technical feedback
- asking questions in good faith
- helping others improve their contributions
- acknowledging mistakes and correcting them
- keeping discussion relevant to the project

## Unacceptable Behavior

Examples of unacceptable behavior include:

- personal attacks, insults, or harassment
- discriminatory language or conduct
- intimidation, trolling, or sustained disruption
- publishing private information without permission
- sexualized language or unwanted attention
- bad-faith technical derailment or abusive review behavior

## Scope

This code of conduct applies in project spaces, including:

- GitHub issues
- pull requests
- discussions
- code review comments
- other project communication channels

## Reporting

If you experience or witness behavior that violates this code of conduct, report it to the maintainer privately.

If GitHub private reporting or moderation tools are enabled for this repository, you may use those when appropriate.

Reports should include:

- what happened
- where it happened
- who was involved
- any relevant screenshots or links

## Enforcement

The maintainer may remove comments, reject contributions, restrict participation, or take other actions needed to keep the project safe and productive.

Enforcement decisions will be made at maintainer discretion and on a best-effort basis.
